# Example

The next question is two-sided. One side is whether the Etherloop migrates into the rest of the lineup. Munro & Associates' 2025 teardown observed that features like steer-by-wire, exportable power, and 48V systems "will soon become staples in other Tesla models," and each of those features depends on the kind of high-bandwidth, redundant network this patent specifies. The Model S and Model 3 still run on older wiring architectures.

The other side is whether the Ethernet spine extends into more layers within each car. Body control, climate, and infotainment subsystems still ride on CAN today, and claim 1 keeps those endpoints CAN-bound by design. Whether Tesla's next iteration moves more of those endpoints onto the ring directly is the architectural question the patent leaves open. 🤔
