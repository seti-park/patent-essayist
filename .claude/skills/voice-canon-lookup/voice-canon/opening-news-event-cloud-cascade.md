# Example

On April 22, 2026, at Cloud Next in Las Vegas, Google Cloud renamed Vertex AI as the Gemini Enterprise Agent Platform. The platform inherited Vertex's developer-facing services and added new agent-orchestration components: Agent Identity, Agent Gateway, Agent Studio, and an Agent Development Kit that processes more than six trillion tokens per month.

Six days later, on April 28, Alphabet reported first-quarter results. Google Cloud revenue had grown 63 percent year-over-year to $20 billion. Operating income had tripled, from $2.2 billion to $6.6 billion. Backlog crossed $460 billion, nearly doubling from the previous quarter.

**Revenue grew 63 percent. Operating income grew 200 percent.** The gap between those numbers describes a routing decision the platform makes billions of times per minute: which model to ask, when to ask it, and when not to ask at all.
