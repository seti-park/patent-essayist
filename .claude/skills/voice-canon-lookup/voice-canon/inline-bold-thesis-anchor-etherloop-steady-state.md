# Example

The Etherloop makes redundancy the default operation. Both directions of the ring carry the same data all the time. There is nothing to switch over to, because nothing was switched away from. **The redundancy is not activated by failure; it is the steady-state behavior of the network.**

The cost is link utilization. Sending each vehicle message in two directions doubles bandwidth use. Claim 5 absorbs that cost selectively: vehicle messages get the doubling, audio does not.
