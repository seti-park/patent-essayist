## What lives here (port the canon)

- The narrator persona / point of view for the essay.
- Sentence rhythm, diction, register, and the signature moves that make the voice itself.
- Worked examples of the voice (before/after), if the canon includes them.

This file is the *intent* of the voice. `deliverable-voice-rules.md` holds the checkable
*rules*, and `anti-ai-writing.md` holds the *prohibitions*. Phase 3 judges compliance from
those two (not this file), so anything that must be enforced at edit time has to be
expressed as a rule there, not only described here.
