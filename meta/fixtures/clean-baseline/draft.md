# Tesla's 70ms Vindication

The rotor of the predictive controller reads the vision sensor before impact. The
controller then commits the deployment decision. This is described plainly in the
disclosure. See [0016] for the mechanism and Figure 1 for the signal path.

The patent claims a 70-millisecond lead [0024]. Figure 2 shows the timing diagram that
makes the comparison concrete.
